# Grafana Nuraya

A community distribution of [Grafana], similar to AMG
([announcement][AMG announcement], [update][AMG update]).

## Build
```shell
git clone https://github.com/nurayadb/grafana-nuraya.git
cd grafana-nuraya
sh build.sh
```

## Usage
```shell
sh run.sh
```

## OCI
```shell
docker pull ghcr.io/nurayadb/grafana-nuraya:latest
```
