Suraya

A blend of Grafana and Marimo, with flavours of AMG and PMM.
An application platform unlike Supabase, but for CrateDB.
Also, a personal data exploration workbench.

CrateDB

The CrateDB SQL warehouse stores any type of data (time series, geospatial, json, vector).

CrateDB Cockpit

Alternative admin panel for the CrateDB SQL warehouse.

Grafana

The best-in-class open and composable observability and data visualization platform.

Marimo

A reactive Python notebook environment with an emphasis on composability and shareability.
